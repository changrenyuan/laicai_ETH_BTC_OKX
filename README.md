# laicai_ETH_BTC_OKX
加密币的量化系统
# 量化交易系统项目记录（来财 & 合伙人）

## 项目宗旨

* 手搓可控、可扩展、可实盘的加密货币量化交易系统
* 不与机构/做市商正面竞争
* 目标：小资金长期生存 + 稳定复利

---

## 合作角色

* **你**：策略方向决策 / 实盘风险承担 / 产品负责人
* **来财（AI）**：

  * 系统架构设计
  * Python 量化工程
  * 策略与风控设计
  * 代码审查与逻辑校验

---

## 技术栈（初定）

* 语言：Python 3.10+
* 交易所：OKX
* 接口：REST + WebSocket
* 数据存储：本地（CSV / SQLite），不依赖云
* 版本管理：GitHub

---

## 当前阶段

**阶段 0：项目立项与仓库规范确定（已确认）**

已确认原则：

* Python 实盘交易代码是项目的**绝对核心**
* GitHub 仓库只存放：

  * 已验证可用的代码
  * 已确认的设计与决策文档
* 所有试验性代码、草稿、失败尝试：

  * **仅存在于本地电脑**
  * 不进入仓库

目的：

* 保持仓库长期干净、可信、可回溯
* 仓库内容 = 可直接复用 / 可直接部署
